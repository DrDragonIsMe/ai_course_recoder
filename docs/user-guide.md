## 首次启动与权限

### 安装

1. 从 Xcode 构建后运行，或直接打开生成的 `RecordCourses.app`。
2. 首次启动时，macOS 会弹出权限申请提示。

### 必须权限

| 权限 | 用途 | 不授权的后果 |
|------|------|--------------|
| **屏幕录制** | 捕获显示器内容 | 无法录制屏幕 |
| **摄像头** | 录制画中画讲师画面 | 无法启用摄像头 |
| **麦克风** | 录制讲解音频 | 无法录制麦克风声音 |
| **辅助功能（Accessibility）** | 按键叠加层需要监听全局键盘事件 | 按键显示失效 |

### 授权步骤

1. 如果主界面顶部出现黄色提示条，点击 **Open Settings**。
2. 在「系统设置 → 隐私与安全性」中找到对应权限，开启 `RecordCourses`。
3. 返回应用，提示条消失后即可开始录制。

> **提示**：修改屏幕录制权限后，建议重新启动应用以确保生效。

## 主界面概览

启动后主窗口从上到下依次分为：

- **标题栏**：显示应用名称、当前状态（Ready / Recording）和 **设置（⚙️）** 按钮。
- **权限提示条**：权限不足时显示。
- **配置区**：可滚动的表单，包含：
  - Capture Mode（捕获模式）
  - Layout（布局）
  - Overlay Settings（叠加层设置）
  - Display（显示器选择）
  - Video（视频设置）
  - Audio（音频设置）
  - Output（输出设置）
- **标注工具栏**：仅在录制时显示。
- **底部状态栏**：显示录制时长与开始/停止按钮。

## 录制流程

### 1. 选择捕获模式

在 **Capture Mode** 区域选择：

- **Full Screen**：录制整个屏幕。
- **Window**：录制单个窗口（如已启用）。
- **Region**：录制自定义区域（如已启用）。

### 2. 选择显示器

在 **Display** 下拉菜单中选择要录制的显示器。多屏环境下请确认选中的是目标屏幕。

### 3. 配置视频与音频

- **Camera**：开启/关闭摄像头画中画。
- **Position / Size**：调整摄像头画面位置与大小。
- **Show Cursor**：是否在最终视频中显示系统光标。
- **Microphone**：是否录制麦克风。
- **Quality / Codec / Frame Rate**：调整画质、编码器与帧率。

### 4. 开始录制

点击底部 **Record** 按钮，状态变为 **Recording** 并开始计时。录制过程中可随时开启标注或调整叠加层（部分设置仅在下一帧生效）。

### 5. 停止录制

点击底部 **Stop** 按钮，或按下 `⌘⇧S`。录制完成后会自动打开保存目录。

## 布局（Layout）

在 **Layout** 区域点击预设卡片切换画面布局：

| 布局 | 说明 | 摄像头 |
|------|------|--------|
| **Full Screen** | 全屏录制，无画中画 | 关闭 |
| **Corner PIP** | 屏幕主体 + 右下角小摄像头窗口 | 开启 |
| **Presenter Left** | 左侧讲师画面 + 右侧屏幕内容 | 开启 |
| **Presenter Right** | 右侧讲师画面 + 左侧屏幕内容 | 开启 |
| **Software Demo** | 纯屏幕录制，突出软件操作 | 关闭 |

> 切换布局时会自动同步摄像头的开启状态。

## 叠加层（Overlays）

在 **Overlay Settings** 区域启用并配置以下叠加层。

### 水印（Watermark）

- 开启后输入 **Logo** 与 **Instructor** 名称。
- 选择显示位置：左下角 / 右下角。
- 水印会始终显示在视频角落。

### 光标高亮（Cursor）

- **Highlight**：以光标为中心显示一个高亮圆环。
- **Click Ripple**：点击鼠标时显示扩散涟漪动画。
- 适合强调鼠标操作位置。

### 按键显示（Keystrokes）

- 开启后会在画面角落显示最近按下的快捷键。
- 选择 **Position**：左下角 / 右下角。
- 最多显示最近 5 个按键组合。

> 需要授权辅助功能权限才能捕获按键。

### 放大镜（Magnifier）

- 开启后会在光标位置显示一个圆形放大镜。
- 镜头内部放大光标周围区域，适合展示细小 UI 元素。
- 放大镜会跟随光标移动。

### 步骤标注（Steps）

- 在配置中开启后，可在录制时通过标注工具添加带编号的步骤标记。
- 每个步骤包含编号、文字说明和目标位置。

### 字幕（Subtitles）

1. 开启 **Subtitles** 开关。
2. 点击 **Import SRT…** 选择 `.srt` 字幕文件。
3. 导入成功后显示条目数量。
4. 录制时字幕会按当前时间自动匹配并显示在画面底部。
5. 开启 **Bilingual** 后，若 SRT 条目内用换行符分隔两种语言，会自动分行显示。

### 章节进度条（Progress）

- 开启后在画面底部显示一条进度条。
- 使用 **Add Chapter** 添加章节，填写标题和时间戳（秒）。
- 进度条会显示录制进度并标记章节位置。

## 实时标注

开始录制后，屏幕顶部会出现悬浮工具栏，提供：

- **录制状态 / 时长**
- **标注开关（🖊 / 🖊✓）**：点击即可进入或退出标注绘制模式，等同于 `⌘⇧A`。
- **停止按钮**

同时主窗口底部也会显示标注工具栏，用于选择具体工具、颜色和粗细。

### 标注工具

| 工具 | 说明 |
|------|------|
| 画笔（Pen） | 自由手绘线条 |
| 箭头（Arrow） | 绘制带箭头的指向线 |
| 矩形（Rectangle） | 绘制矩形框 |
| 圆形（Circle） | 绘制圆形/椭圆框 |
| 橡皮擦（Eraser） | 擦除已有笔画 |

### 标注颜色

工具栏提供红、橙、黄、绿、蓝、紫、白、黑八种颜色，点击切换。

### 线条粗细

工具栏提供 2 / 4 / 6 / 10 四种粗细。

### 操作

- **Undo**：撤销上一步标注。
- **Redo**：重做。
- **Clear all**：清空所有标注。

## 输出与文件管理

### 默认保存位置

录制完成后，视频默认保存到：

```
~/Movies/RecordCourse-YYYYMMDD-HHMMSS.<mov|mp4>
```

### 设置输出格式

在 **Output** 区域选择：

- **MOV**（默认，兼容性最好）
- **MP4**

### 打开保存目录

停止录制后，应用会自动打开 Finder 并定位到保存目录。

## 常见问题

### Q1: 点击 Record 后没有反应？

- 检查是否已授予**屏幕录制**权限。
- 检查是否已选择有效的显示器。

### Q2: 摄像头画面不显示？

- 确认 **Camera** 开关已打开。
- 确认布局预设不是「Full Screen」或「Software Demo」。
- 检查系统设置中的摄像头权限。

### Q3: 按键显示不出来？

- 按键叠加层依赖**辅助功能**权限，请到「系统设置 → 隐私与安全性 → 辅助功能」中授权。
- 仅当焦点在其他应用（被录制的窗口）时全局按键监听才会生效。

### Q4: 字幕导入后没有显示？

- 确认 **Subtitles** 开关已打开。
- 确认 SRT 文件编码为 UTF-8。
- 检查录制时间是否已到达字幕条目的开始时间。

### Q5: 录制的视频没有标注？

- 标注仅在录制过程中绘制，且需要进入标注模式（`⌘⇧A` 或点击标注工具栏）。
- 确认开始录制后再进行标注。
